What is Version Control System (VCS)?

Version Control Systems are the software tools for tracking/managing all the changes made to the source code during the project development. It keeps a record of every single change made to the code. It also allows us to turn back to the previous version of the code if any mistake is made in the current version. Without a VCS in place, it would not be possible to monitor the development of the project.

What is Git?

Git is a version control system which lets you track changes you make to your files over time.
With Git, you can revert to various states of your files (like a time traveling machine). You can also make a copy of your file, make changes to that copy, and then merge these changes to the original copy.

Installation of Git

In order to use Git, you have to install it on your computer. To do this, you can download the latest version on the official website.
After installation, you can verify the current version of git installed on you PC, by running following command on command line:

git --version
.
The next thing you'll need to do is to set your username and email address. Git will use this information to identify who made specific changes to files.
To set your username, type and execute these commands:
git config --global user.name "YOUR_USERNAME"
git config --global user.email "YOUR_EMAIL"
